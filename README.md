Sails
=====

Sails, create [Thrift](thrift.apache.org) app server like Rails.

## Installation

```
$ gem install sails
```

## Usage

### Create new project

```
$ sails new foo
$ cd foo
$ sails start
```

### Generate Thrift IDL

You can edit Thrift IDL in `app_name.thrift`, and then generate it to ruby source code.

```
$ rake gen
```
