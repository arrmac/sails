Tails
=====

Tails, create [Thrift](thrift.apache.org) app server like Rails.

## Installation

```
$ gem install tails
```

## Usage

### Create new project

```
$ tails new foo
$ cd foo
$ tails start
```

### Generate Thrift IDL

You can edit Thrift IDL in `app_name.thrift`, and then generate it to ruby source code.

```
$ rake gen
```
